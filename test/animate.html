CSS Animations

Multiple transitions

Animations tween between css styles. Can be set for multiple changes.

@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

@keyframes example {
    0%   {background-color: red;}
    25%  {background-color: yellow;}
    50%  {background-color: blue;}
    100% {background-color: green;}
}

Animation Properties

Properties of the animation can be set on mutliple lines. Duration is required for the animation to show, the other properties will set to defaults.

div {
	animation-name: example;
	animation-duration: 5s;

	animation-timing-function: ease;
		/*
		ease - slow start, then fast, then end slowly
		linear - flat speed
		ease-in - slow start
		ease-out - slow end
		ease-in-out - slow start and end
		cubic-bezier(n,n,n,n) - lets you define your own values in a cubic-bezier function
		*/
	animation-delay: 2s;

	animation-iteration-count: infinite;
	animation-direction: alternate;
		/*
		reverse - run the animation backwards
		alternate - run the animation normally, then backwards, etc
		*/
}
div {
    animation: example 5s ease 2s infinite alternate;
}
